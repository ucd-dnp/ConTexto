1. Instalación ConTexto¶

1.1. Instalación básica¶

Para la instalación de la librería se recomienda utilizar el gestor de paquetes pip. Por buenas prácticas se sugiere antes de la instalación crear un entorno virtual que permita aislar las librerías y así evitar conflictos de versiones con el entorno de desarrollo base del computador. Esta instalación no presenta restricciones de arquitectura, es decir, funciona tanto para arquitecturas de 32 bits como arquitecturas de 64 bits.

pip install contexto

De manera alterna también puede utilizar el gestor de paquetes conda; sin embargo, dado que algunos paquetes no se encuentran disponibles en este gestor será necesario realizar la instalación manualmente de los paquetes librería_1, librería_2 y librería_3.

conda install -c ucd-dnp contexto

1.2. Instalación completa¶

Otra opción de instalación corresponde a utilizar los archivos compilados wheel que se encuentran cargados en el repositorio de GitHub de la librería. Al utilizar los archivos wheel se instalarán las dependencias de ConTexto, equivalente a utilizar el comando pip install contexto y adicionalmente se instalará el framework PyTorch (con instalación para uso de CPU) y el paquete Stanza, estos últimos se utilizan en el módulo de Lematización, sin embargo, su instalación no es de caracter obligatorio, ya que por defecto se utiliza el lematizador de spaCy.

Advertencia

  • La instalación utilizando los archivos wheel solo soportan Python en arquitecturas de 64 bits (x64) para Windows. Para Linux no se presenta esta restricción.
  • Versión máxima soportada de Python v3.8.5
  • Versión mínima soportada de Python v3.6.1
pip install contexto -f https://ucd-dnp.github.io/ConTexto/contexto_stable.html
